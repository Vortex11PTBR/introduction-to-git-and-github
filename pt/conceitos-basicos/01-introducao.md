# Introdução ao Git

Git é um **sistema de controle de versão distribuído**. Sua principal função é rastrear alterações em arquivos, especialmente em projetos de desenvolvimento de software. Ele permite que múltiplos desenvolvedores trabalhem de forma colaborativa e organizada.

Com o Git, é possível:

- Manter um histórico completo de todas as modificações.
- Reverter arquivos ou o projeto inteiro para um estado anterior.
- Criar ramificações (branches) para desenvolver novas funcionalidades de forma isolada.
- Comparar e mesclar (merge) diferentes versões de um mesmo arquivo.
- Trabalhar de forma offline, sincronizando as alterações posteriormente com um repositório remoto.
