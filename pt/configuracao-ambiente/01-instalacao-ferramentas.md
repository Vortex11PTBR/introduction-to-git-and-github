# Instalação de Ferramentas Essenciais

Para um ambiente de desenvolvimento completo e eficiente com Git, é recomendável instalar as seguintes ferramentas:

- **Git Bash**: Um terminal que emula o ambiente Linux no Windows, incluindo o Git e comandos Unix básicos. Essencial para usuários Windows.
- **Virtual Box**: Software de virtualização que permite rodar sistemas operacionais (como Ubuntu) dentro do seu sistema operacional principal.
- **Ubuntu (Linux)**: Uma distribuição Linux popular, ideal para desenvolvimento, que pode ser instalada em uma máquina virtual via Virtual Box.

## Onde Instalar

Recomenda-se baixar e instalar essas ferramentas diretamente dos **sites oficiais** de cada projeto para garantir as versões mais recentes e seguras.

## Configuração do Editor de Código (VS Code)

Para uma melhor experiência com arquivos Markdown e desenvolvimento em geral, o **Visual Studio Code (VS Code)** é altamente recomendado. Após a instalação:

1. Instale a extensão `zaaack.markdown-editor`.
2. Utilize `Ctrl + Shift + P` (ou `Cmd + Shift + P` no macOS) e procure por "Markdown: Open Preview to the Side" para visualizar as edições do seu arquivo Markdown em tempo real.
